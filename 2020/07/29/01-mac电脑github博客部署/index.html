|_config.yml |网站的配置文件，可以在此配置大部分参数 |
|source |资源文件夹是存放用户资源的地方。除source下的 _posts 文件夹之外，开头命名为 _ (下划线)的文件 / 文件夹和隐藏的文件将会被忽略。Markdown 和 HTML 文件会被解析并放到 public 文件夹，而其他文件会被拷贝过去。 |
